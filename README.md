# Task Tracker CLI
A simple command line interface (CLI) to track and manage your tasks, built with Python.

## How to Run
Make sure you have Python installed, then navigate to the project folder.

## Available Commands
| Command | Description |
|---|---|
| `py task_cli.py add "Task"` | Add a new task |
| `py task_cli.py update 1 "New"` | Update a task |
| `py task_cli.py delete 1` | Delete a task |
| `py task_cli.py mark-done 1` | Mark as done |
| `py task_cli.py mark-in-progress 1` | Mark as in progress |
| `py task_cli.py list` | List all tasks |
| `py task_cli.py list done` | List by status |
